About the Project:

I made this Discord bot back in 2021 when discord was cracking down on common discord music bots.

I didn't want to have to keep finding a new music bot, so I decided it'd be a fun project to try and make my own.

I used Python and Replit to host the bot because there was already a decent amount of documentation on the topic.

It was a very fun project and I learned some new things, such as how to host a program online, how to deal with .env files, and how to make API calls.
