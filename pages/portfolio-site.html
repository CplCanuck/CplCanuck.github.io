About the Project:

This website was the final project for my "COMP2511 - Web I" class at Mount Royal University.

It was interesting to write an entire website with just bare metal HTML and CSS, but I like doing projects like this.
It's like learning C or assembly, it give a good foundation so you understand what going on under the hood when you do more abstract things.

Probably the biggest challenge with this project was just finding inspiration for what I wanted to do for the website design.
