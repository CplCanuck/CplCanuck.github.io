About the Project:

Git Repo

I made this project to learn the SMFL C++ graphics library on.

If I recall correctly, when I was writing this project, I tied each value directly to visuals, which was a bit of a challenge to work around.

If I were to redo or refactor this project, decoupling the values from the visuals would be one of the first things I'd do. I would also try to implement a visually different sorting algorithm, like MergeSort or QuickSort.
